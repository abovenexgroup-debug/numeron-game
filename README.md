# 🔢 数字当てゲーム（ヌメロン）

3桁の数字を当てる脳トレゲームです。

## 遊び方

1. 0〜9の数字から、隠された3桁の数字を当てます
2. 回答すると、ヒントがもらえます
   - ◎ ピッタリ：数字も場所も正解
   - ○ おしい：数字はあるけど場所が違う
3. ヒントを頼りに正解を目指そう！

## 難易度

- 😃 ふつう（3ケタ）
- 😤 むずかしい（4ケタ）
